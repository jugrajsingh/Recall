---
title: Commands
description: Recall command reference
---

## Sync

```bash
recall sync
recall sync --force
```

## Search

```bash
recall
recall search "query"
recall search "query" --project /path/to/repo
```

## Usage

```bash
recall usage
recall usage --json
```

## Export

```bash
recall export --jsonl --source codex --project /path/to/repo --limit 20
```

## Info

```bash
recall info
```
